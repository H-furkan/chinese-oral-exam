# 🀄 Chinese Oral Exam Trainer

Interactive study site for the Chinese final oral exam (lessons 6–10).

**Live site:** https://h-furkan.github.io/chinese-oral-exam/

## What's inside

- **Part 1 – Vocabulary flashcards:** 125 English → Chinese cards with pinyin and progress tracking
- **Part 2 – Q&A practice:** read the question in characters only (simulates listening), answer out loud, then reveal
- **Part 3 – Make a question:** see an answer, produce the question; includes a question-word cheat sheet
- **Part 4 – Monologues:** the four speaking topics (weekly plan, friend, wallet, room) with model answers and a recite mode
- **Dialogue bank:** all 39 practice Q&As with pinyin/English toggles and hide-answer self-testing
- **Quiz:** randomized 10-question mixed quiz with best-score tracking

No build step — plain HTML/CSS/JS.
